⚡ ProSport | Sports Ecommerce

ProSport is a modern and responsive sports ecommerce web app with a clean dark UI and smooth user experience.

🚀 Features

Home, Shop, Product, Cart & Order History pages

Add to cart, update quantity, remove items

Product search, filter & sorting

Responsive navbar & mobile menu

Toast notifications for actions

Cart data saved using localStorage

🛠️ Tech Stack

React 19 + Vite

Tailwind CSS

React Router DOM

Lucide React Icons
